Die Trennung von Demo- und Vollversion-Speicherdaten kann verwirrend sein, ist aber darauf ausgelegt, einen Vorgeschmack auf das Spiel zu geben und Spieler gleichzeitig dazu zu motivieren, sich mit dem vollständigen Erlebnis zu beschäftigen. Hier ist eine Aufschlüsselung dessen, was Sie erwarten können:

| Funktion          | Demo-Version                                | Vollversion                                            |
| :---------------- | :------------------------------------------ | :----------------------------------------------------- |
| **Speicherdaten** | **Wird NICHT übertragen**                   | Permanenter Fortschritt, auf Ihrem Konto gespeichert   |
| **Quests**        | Begrenzter Inhalt                           | **Mehr Quests**, tiefere Handlungsstränge              |
| **Orte**          | Eingeschränkte Bereiche                       | **Mehr Orte** in Evalia zu erkunden                   |
| **Charaktere**    | Weniger NPCs, grundlegende Interaktionen    | **Mehr Charaktere**, komplexe Beziehungen, Entscheidungen |
| **Gegenstände**   | Grundlegende Überlebensgegenstände          | **Mehr Gegenstände**, Waffen, Werkzeuge, Handwerksmaterialien |
| **Basenbau**      | Nicht verfügbar oder sehr begrenzt         | **Vollständiges Basenbausystem**                       |
| **Minispiele**    | Nicht verfügbar oder begrenzt              | **Enthält verschiedene Minispiele**                     |
| **Level-Up-Attribute** | Grundlegende Attribute                  | **Mehr Level-Up-Attribute** für Charakterentwicklung   |
| **Belohnung**     | **Kostenlose Kristalle** nach Abschluss    | Verdienen Sie Belohnungen durch Quests, Erkundung und Events |
